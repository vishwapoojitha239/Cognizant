# HOL 03 - Score Calculator App

## Aim
Create a React app using props to display student score details and calculate percentage.

## Features
- `CalculateScore` functional component
- Accepts `name`, `school`, `total`, `goal` as props
- Calculates percentage and grade
- Color-coded grades with CSS styling

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
Student cards showing name, school, total marks, marks obtained, percentage, and grade.
