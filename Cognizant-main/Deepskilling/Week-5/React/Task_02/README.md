# HOL 02 - Student App

## Aim
Create a multi-component React app with Home, About, and Contact pages for a Student Management Portal.

## Features
- Three functional components: Home, About, Contact
- Navigation buttons to switch between pages
- Each page displays its respective welcome message

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
- Home: "Welcome to the Home page of Student Management Portal"
- About: "Welcome to the About page of Student Management Portal"
- Contact: "Welcome to the Contact page of Student Management Portal"
