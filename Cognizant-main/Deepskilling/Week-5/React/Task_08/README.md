# HOL 08 - Counter App

## Aim
Create a class component to track people entry and exit using state management.

## Features
- `CountPeople` class component
- State: `entrycount` and `exitcount`
- Login button increments entrycount
- Exit button increments exitcount
- Displays current people inside (entry - exit)

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
A counter dashboard with Entry/Exit buttons and live count display.
