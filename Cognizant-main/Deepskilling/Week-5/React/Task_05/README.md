# HOL 05 - Cohort Styling App

## Aim
Create a React app using CSS Modules to style cohort detail cards.

## Features
- `CohortDetails` functional component
- `CohortDetails.module.css` with scoped styles
- `.box` class: width 300px, inline-block, margin 10px, padding 10px 20px, 1px solid black border, border-radius 10px
- `dt` font-weight 500
- Ongoing cohort h3 in green, others in blue

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
Cohort cards with scoped CSS module styles. Green heading for ongoing, blue for completed.
