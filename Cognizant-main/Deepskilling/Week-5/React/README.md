# ReactJS Hands-On Labs (HOL)

This folder contains all 19 ReactJS Hands-On Lab tasks from the Cognizant Deepskilling program.

## Structure

| # | Folder | Topic |
|---|--------|-------|
| 01 | myfirstreact | First React App |
| 02 | StudentApp | Multiple Components |
| 03 | scorecalculatorapp | Props & Score Calculation |
| 04 | blogapp | Class Components & Lifecycle |
| 05 | cohort_styling_app | CSS Modules |
| 06 | TrainersApp | React Router DOM |
| 07 | shoppingapp | Class Component & State |
| 08 | counterapp | State Management |
| 09 | cricketapp | ES6 Features |
| 10 | officespacerentalapp | JSX & Objects |
| 11 | eventexamplesapp | Event Handling |
| 12 | ticketbookingapp | Conditional Rendering |
| 13 | bloggerapp | Conditional & List Rendering |
| 14 | context_api_employee_app | Context API |
| 15 | ticketraisingapp | Forms & Refs |
| 16 | mailregisterapp | Form Validation |
| 17 | fetchuserapp | API Fetching |
| 18 | cohort_unit_testing_app | Unit Testing |
| 19 | gitclientapp | Axios & Testing |

## Tech Stack
- React 18
- Vite
- react-router-dom (HOL 06)
- axios (HOL 19)
- Vitest + React Testing Library (HOL 18 & 19)

## How to Run Any App

```bash
cd <folder-name>
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.
