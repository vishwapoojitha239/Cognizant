# HOL 07 - Shopping App

## Aim
Create a class-based React component to display a shopping cart with items.

## Features
- `Cart` class with `itemname` and `price`
- `OnlineShopping` class component
- 5 shopping items initialized in constructor state
- Loop with `.map()` to display item name and price
- Total price calculation

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
A shopping cart table with 5 items showing item name, price, and total.
