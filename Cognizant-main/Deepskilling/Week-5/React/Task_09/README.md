# HOL 09 - Cricket App

## Aim
Demonstrate ES6 features in React: map, filter, destructuring, spread operator, and conditional rendering with if-else.

## Features
- `ListofPlayers`: 11 players with scores, ES6 map, filter for score < 70
- `IndianPlayers`: destructuring for odd/even teams, spread to merge T20 + Ranji arrays
- Flag variable with if-else to toggle between components

## How to Run
```bash
npm install
npm run dev
```
Open `http://localhost:5173`

## Output
Toggle between player list (with low-scorer filter) and Indian players (with merged squad).
